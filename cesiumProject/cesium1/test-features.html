3D模型功能测试

1. 模型隐藏显示功能测试

测试步骤：
  1. 打开应用程序：http://localhost:8081
  2. 在模型列表中，尝试勾选/取消勾选不同的模型
  3. 观察3D场景中对应的模型是否显示/隐藏
预期结果：勾选模型时，3D场景中对应的模型应显示；取消勾选时，模型应隐藏。

2. 全局透明度控制功能测试

测试步骤：
  1. 打开应用程序：http://localhost:8081
  2. 拖动全局透明度滑块
  3. 观察3D场景中所有可见模型的透明度是否变化
预期结果：拖动滑块时，所有可见模型的透明度应按比例变化。

3. 单个模型透明度控制功能测试

测试步骤：
  1. 打开应用程序：http://localhost:8081
  2. 在模型列表中，尝试调整单个模型的透明度滑块
  3. 观察3D场景中对应的模型透明度是否变化
预期结果：调整滑块时，对应的模型透明度应按比例变化。

4. 模型高亮功能测试

测试步骤：
  1. 打开应用程序：http://localhost:8081
  2. 在模型列表中，点击"高亮"按钮
  3. 观察3D场景中对应的模型是否高亮显示
  4. 等待3秒后，观察模型是否恢复原来的颜色
预期结果：点击高亮按钮时，模型应变为黄色；3秒后恢复原来的颜色。

5. 显示/隐藏所有模型功能测试

测试步骤：
  1. 打开应用程序：http://localhost:8081
  2. 点击"显示所有"按钮
  3. 观察3D场景中所有模型是否显示
  4. 点击"隐藏所有"按钮
  5. 观察3D场景中所有模型是否隐藏
预期结果：点击"显示所有"时，所有模型应显示；点击"隐藏所有"时，所有模型应隐藏。
