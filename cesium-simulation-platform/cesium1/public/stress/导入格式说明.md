# 应力分析导入说明

## 设计原则

- 系统按字段结构自动识别数据形态
- 不使用额外标识进行分流
- 导入后统一进入同一条重建与渲染链路

## 支持的数据形态

| 数据形态 | 关键字段 | 说明 |
| --- | --- | --- |
| 点序列 | `点` 或 `points` | 每个点包含坐标和应力时间序列 |
| 点位帧 | `点位 + 时间点` | 点定义与时间帧拆分，适合监测点模式 |
| 网格场 | `网格 + 数据` | 体素场应力分布 |

## 通用必备信息

- 坐标原点：`[经度, 纬度, 高度]` 或 `{ 模式: "模型中心" }`
- 时间信息：`时间点` 或 `时间.帧数`
- 应力数值：`von_mises` 或六分量张量
- 场尺寸：点使用 UVW/ENU 时建议提供 `场尺寸`

## 点序列示例

```json
{
  "坐标原点": { "模式": "模型中心" },
  "场尺寸": [200, 200, 100],
  "时间": { "维度": "秒", "时间点": [0, 1, 2] },
  "点": [
    {
      "id": "P01",
      "中心_ENU_m": [10, 20, 30],
      "von_mises": [12.5, 14.3, 15.1]
    }
  ]
}
```

## 点位帧示例

```json
{
  "坐标原点": { "模式": "模型中心" },
  "场尺寸": [200, 200, 100],
  "点位": [{ "id": "P01", "中心_ENU_m": [10, 20, 30] }],
  "时间点": [
    { "t": 0, "点": [{ "id": "P01", "von_mises": 12.5 }] },
    { "t": 1, "点": [{ "id": "P01", "von_mises": 14.3 }] }
  ]
}
```

## 网格场示例

```json
{
  "坐标原点": [110, 30, 0],
  "场尺寸": [100, 100, 50],
  "网格": { "宽": 2, "高": 2, "深": 2 },
  "时间": { "维度": "秒", "帧数": 1 },
  "材料": {
    "弹性模量E": 30000,
    "泊松比nu": 0.25,
    "单轴抗压强度_UCS": 100
  },
  "数据": {
    "类型": "应力",
    "帧": [
      {
        "xx": [1, 2, 3, 4, 5, 6, 7, 8],
        "yy": [1, 2, 3, 4, 5, 6, 7, 8],
        "zz": [1, 2, 3, 4, 5, 6, 7, 8],
        "xy": [0, 0, 0, 0, 0, 0, 0, 0],
        "yz": [0, 0, 0, 0, 0, 0, 0, 0],
        "zx": [0, 0, 0, 0, 0, 0, 0, 0]
      }
    ]
  }
}
```

## 常见错误

- 无法识别导入数据：请补齐 `点/点位+时间点` 或 `网格+数据`
- 坐标原点格式错误：请改为数组或模型中心模式
- 时间长度不一致：应力序列长度需与时间帧一致
